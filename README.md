# NautilusVision
Team Nautilus particiaptes at Robosub and represents the University of Sherbrooke. We are 8 students working on this project as our capstone project

The repo is split into two halves, on the first hand, we have the datasets folder containing all the images used on this repo. They are separated by time taken and some of them include depth maps when we took them with our oak s2 -d.

The other half contains all the successful and failed attemps to make a functional algorithm for our competition.
